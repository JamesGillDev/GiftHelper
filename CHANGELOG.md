# Changelog

All notable changes to this project are documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.2.0] - 2026-02-25
### Added
- Local gift suggestion service for profile-based recommendation generation.
- Gift Finder and shortlist-focused iteration for the public release flow.

### Changed
- Public release versioning now standardized with SemVer across project artifacts.

## [1.1.0] - 2026-02-25
### Added
- Render deployment blueprint.
- Docker-based deployment path with persistent SQLite support.

### Changed
- Release/deployment guidance for public hosting iterations.

## [1.0.1] - 2026-02-25
### Added
- Downloadable release artifact link in project documentation.

### Changed
- README updates for public repo and release references.
- Ignored local artifact output for cleaner release packaging.

## [1.0.0] - 2026-02-25
### Added
- Initial public release of Gift Helper App.
